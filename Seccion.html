Etiquetas de Titulo

La etiqueta <h#> se utiliza para crear titulos, Un numero mas grande (6) equivale a un tamaño de titulo menor .

  • Titulo <h1>

  • Titulo <h2>

  • Titulo <h3>

  • Titulo <h4>

  • Titulo <h5>
  • Titulo <h6>

titulos en grupos o <hgrup>

El elemento <hgroup> en HTML se utiliza para agrupar uno o más elementos de encabezado, como un título principal y un subtítulo, o varios encabezados relacionados. Esto ayuda a organizar mejor la estructura del documento, indicando que los encabezados dentro de ese grupo están relacionados entre sí.

Etiqueta <Address>

El elemento <Address > en HTML se utiliza para definir la información de contacto del autor o propietario de un documento o artículo.

Autor: Francisco Muñoz Castro

Correo electrónico: francisco@example.com

Teléfono: +506 1234-5678

Dirección: Calle 123, Ciudad, País

Si agregas en la parte del correo un <href> con un "mailto" puedes mandar un correo directamente al propietario
