Etiquetas de estructura básica:

<html>

La etiqueta <html> define el inicio y el final de un documento HTML.

<head>

La etiqueta <head> contiene metadatos sobre el documento, como el título y enlaces a hojas de estilo.

<body>

La etiqueta <body> incluye el contenido visible de la página, como texto, imágenes y enlaces.

Ejemplo de Código:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Página Web</title>
</head>
<body>
  <h1>Bienvenido a mi página web</h1>
  <p>Este es un párrafo de texto.</p>
</body>
</html>
