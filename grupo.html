Etiqueta <ul>

La etiqueta <ul> define una lista desordenada en HTML. Los elementos de la lista se colocan dentro de etiquetas <li>, que representan cada ítem de la lista.

  • Elemento de la lista 1
  • Elemento de la lista 2
  • Elemento de la lista 3
  • Elemento de la lista 4

Etiqueta <ol>

La etiqueta <ol> define una lista ordenada en HTML. Los elementos de la lista se colocan dentro de etiquetas <li>, que representan cada ítem de la lista y están numerados automáticamente en orden.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. Cuarto elemento

Etiqueta <li>

La etiqueta <li> define un elemento dentro de una lista en HTML. Puede contener texto, imágenes, enlaces y otros elementos HTML.

<ol>
<li>Primer elemento</li>
<li>Segundo elemento</li>
<li>Tercer elemento</li>
<li>Cuarto elemento</li>
</ol>

Etiqueta <dl>

La etiqueta <dl>define una lista de descripción en HTML. Esta lista contiene términos y sus respectivas descripciones.

HTML
Lenguaje de marcado utilizado para estructurar contenido en la web.
CSS
Lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML.
JavaScript
Lenguaje de programación utilizado para crear interactividad en páginas web.
